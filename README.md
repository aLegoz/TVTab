# TVTab

Настільний застосунок для обліку робочого часу (форма Т-13) та розрахунку зарплат.

## Завантаження

👉 [Releases](https://github.com/aLegoz/TVTab/releases) — скачати `TVTab-*-portable-win64.zip`, розпакувати, запустити `TVTab.exe`. Встановлення не потрібне.

---

## Коди відвідуваності

| Код | Значення |
|-----|----------|
| Я   | Явка (відпрацьований день) |
| В   | Вихідний |
| О   | Відпустка |
| Б   | Лікарняний |
| ОЗ  | Відпустка за свій рахунок |
| Н   | Неявка (поважна причина) |
| НН  | Прогул |

## Розрахунок зарплати

### Типи ставок
- **Оклад (погодинний облік)** — місячна ставка, розраховується на основі відпрацьованих годин та норми місяця
- **Погодинна** — ставка × відпрацьовані години

### Коефіцієнти (налаштовуються на кожен місяць окремо)
- **Переробіток** (за замовчуванням 1.5) — множник для годин понад норму
- **Відпустка** (за замовчуванням 1.0) — частина оклада за дні відпустки
- **Лікарняний** (за замовчуванням 0.0) — частина оклада за лікарняні дні

## Режими роботи

### Локальний (serverless)
База даних SQLite зберігається локально на комп'ютері. Не потрібне з'єднання з мережею.

**Розташування БД:**
- Windows: `%APPDATA%\tvtab\companies\`

### Серверний
Підключення до REST API сервера. Дозволяє працювати з одних даних з кількох комп'ютерів.

Вкажіть URL сервера в **Налаштування → Сервер**.

> Серверна частина (Node.js/Express) знаходиться в папці `server/` (готується).
